# Restaurant Management System

A backend restaurant management system developed as part of the CodeAlpha Backend Development Internship.

## Features

* Menu Management
* Order Processing
* Table Reservation
* Inventory Tracking
* REST API Support
* SQLite Database Integration

## Tech Stack

* Python
* Flask
* SQLite

## Installation

1. Clone the repository

git clone <repository-url>

2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py

## API Endpoints

* GET /menu
* POST /order
* GET /orders
* POST /reservation
